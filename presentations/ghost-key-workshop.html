Technical Primer · 1 of 3

How a wave
carries a secret

Radio is energy wiggling at a frequency. To send data, you stamp a pattern onto that wiggle — that's modulation. The simplest kind is On-Off Keying (OOK): signal on = 1, signal off = 0.

That on/off pattern is the ENTIRE secret. The captured "OPEN" code is just a list of ons and offs — which is exactly why copying it is easy.

The vocabulary

Wave cycles per second (Hz). Picks the "channel."

Physical length of one cycle. Higher freq = shorter.

Strength of the wave — how loud / how far.

The base frequency you broadcast on.

How you write data onto the carrier.
OOK / ASK
On = 1, off = 0. Dead simple, very replayable.
FSK
Flip between two frequencies for 0/1. Sturdier.

Bits per second pushed through the air.
ISM bands
License-free frequencies: 433 / 868 / 915 MHz.
RSSI / dBm
How strong a signal you caught.
Antenna
Turns electricity ↔ radio waves. Tuned to a band.
Preamble
A known header so the receiver locks on.

How A Wireless Key Works

Press. Transmit.
Check. Unlock.

A fob or garage remote shouts a code; the receiver checks it and acts. The whole secret is that code — and it comes in two flavors:

Fixed code — the weak one

The same code every single time.

7A3F7A3F7A3F
Rolling code — the strong one

A new code every press, synced to the receiver.

7A3F91C24E0B
Remember which is which. It's about to decide whether your attack works.

Why Your Car Shrugs This Off

Rolling codes
kill the replay

Replay a recorded car-fob code and nothing happens — the car already rolled to the next code. Your recording is spent. That one idea protects nearly every modern vehicle.

  • Each press = a fresh code, never reused.
  • A captured code dies the moment it's seen.
  • (Pros defeat it with fancier tricks — but plain replay is dead.)
Our vault uses a FIXED code. That is the door you're about to walk through.

Factor 2 · Somewhere You Are

How anything knows where you are

Measure your distance from three known points and exactly one spot fits all three. That's trilateration — the engine behind GPS, cell towers, Wi-Fi positioning, Bluetooth beacons, and AirTags.

Three signals + three distances = one location. Simple geometry, enormous reach.
The Math, Walked Through

From distances
to a dot

Each anchor at (xᵢ, yᵢ) measures a distance dᵢ to you. "Distance dᵢ from a point" is just a circle:

(x − xᵢ)² + (y − yᵢ)² = dᵢ²one equation per anchor — but the squares make it ugly
The trick: subtract one circle from another. The x², y² terms cancel.
2(x₂−x₁)x + 2(y₂−y₁)y = (d₁²−d₂²) + (x₂²−x₁²) + (y₂²−y₁²)…and the same for anchors 1 & 3 → two clean LINEAR equations
Two linear equations, two unknowns (x, y). Solve → that's your position. Where do the distances come from? Next slide.
