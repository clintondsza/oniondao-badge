# OnionDAO Badge — Beginner Guide

No hardware experience needed. This guide gets you from box to badge in a few minutes.

---

## What is this?

The OnionDAO Badge is a wearable computer you can hold in your hand. It has a small
black-and-white screen, six buttons, and a chip that can store secret keys. You can
put your own artwork on the screen, navigate menus, and learn about the hardware and
code that makes it work — all from the badge itself.

---

## Powering on

1. Connect the USB-C cable to the badge and to your computer (or any USB power source).
2. Flip the **ON/OFF** slide switch on the back of the board to the ON position.
3. The screen will flash white, then show either your custom home image or the badge's
   name and chip specs.

If the badge goes dark after a minute, it went to sleep — press any button to wake it.

---

## The six buttons


                                             [ CANCEL ]
                             [ UP ]  
                    [ LEFT ]        [ RIGHT ]  
                            [ DOWN ]
                                                   [ SELECT ]
```

| Button | What it does |
|--------|-------------|
| UP / DOWN | Move the menu cursor up or down |
| SELECT | Open the highlighted menu item |
| CANCEL | Go back to the previous screen |
| LEFT / RIGHT | Flip pages in the Hardware or Software Guide |

---

## Navigating the menu

Press any button from the home screen to open the main menu. You will see seven items:

| Menu item | What it shows |
|-----------|---------------|
| 1. System Info | The chip model, clock speed, and memory |
| 2. Button Test | Watch which button you press light up in real time |
| 3. I2C Scanner | Lists every chip the badge found on its internal bus |
| 4. RNG / Crypto | 16 random bytes generated by the hardware crypto chip |
| 5. Display Test | Four graphics patterns to confirm the screen is working |
| 6. Hardware Guide | 10 short lessons about the physical components |
| 7. Software Guide | 10 short lessons about the C++ code that runs the badge |

Use UP / DOWN to move the cursor, SELECT to enter, and CANCEL to go back.

## Putting your own art on the badge

You will need Python 3 installed on your computer.

1. Install the dependencies (one time only):
   ```
   pip install pillow pyserial
   ```

2. Connect the badge via USB and make sure it is awake (press any button if the
   screen is dark).

3. Open the art tool:
   ```
   python3 badge-art/badge_art.py
   ```

4. Draw something, or click **Open Image** to import any photo or PNG.
   Click **Send to Badge** when you are ready.

The badge saves your image to internal flash memory. It will show your art every time
it powers on, even after the battery runs out.

---

## Deep sleep

After 60 seconds of no activity the badge goes to sleep. During sleep it draws almost
no power, and the image on the screen stays visible without any electricity — that is
one of the special properties of e-paper displays.

Press any button to wake the badge back up.

## What is on the board?

The badge is packed with chips. Open the interactive explorer to click on any component
and read what it does:

[docs/board-explorer.html](board-explorer.html) — open this in a browser

Run a local server first so images load correctly:

```
python3 -m http.server 8080
```

Then open: `http://localhost:8080/docs/board-explorer.html`
